# TicoAutos Backend

Backend del proyecto final `TicoAutos` para el curso ISW-711.

## Estado actual

Este repositorio contiene la estructura base del backend. Las siguientes fases agregarán:

- Configuración de Express
- Variables de entorno
- Conexión a MongoDB con Mongoose
- JWT y autenticación
- API REST para vehículos, preguntas y respuestas

## Estructura

```text
src/
  app.js
  server.js
  config/
  models/
  controllers/
  routes/
  middlewares/
  validators/
  utils/
```
